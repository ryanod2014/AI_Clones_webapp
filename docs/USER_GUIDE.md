# AI Clone Video Generator - User Guide

## Getting Started

### Step 1: Get Your API Keys

This application requires API keys from two services:

#### Kie.ai API Key (for image and video generation)
1. Go to [kie.ai](https://kie.ai)
2. Create an account or sign in
3. Navigate to your Dashboard
4. Find your API key in the API section
5. Copy the key

#### ElevenLabs API Key (for voiceover generation)
1. Go to [elevenlabs.io](https://elevenlabs.io)
2. Create an account or sign in
3. Click on your profile icon
4. Select "Profile + API key"
5. Copy your API key

### Step 2: Configure the App

1. Open the AI Clone Video Generator
2. Click the **Settings** icon (gear) in the top right
3. Paste your API keys into the respective fields
4. Click **Save Settings**

> **Note**: Your API keys are stored locally in your browser. They are never sent to our servers.

## Creating Your First Video

### Step 1: Write Your Script

1. Navigate to **Step 1: Script**
2. Enter the message you want your AI clone to speak
3. Tips for great scripts:
   - Keep sentences short and conversational
   - Use punctuation for natural pauses
   - Write how you would speak
   - Avoid complex technical terms

### Step 2: Choose Your Scene

1. Navigate to **Step 2: Scene**
2. You have two options:

**Option A: Use an Existing Scene**
- Browse your saved scenes in the **My Scenes** tab
- Click on a scene to select it

**Option B: Generate a New Scene**
- Click the **Generate New** tab
- Choose orientation (vertical for TikTok/Reels, horizontal for YouTube)
- Optionally upload a reference image
- Describe how you want your AI avatar to look
- Click **Generate Scene**
- If satisfied, click **Use This Scene**

### Step 3: Generate Voiceover

1. Navigate to **Step 3: Voiceover**
2. Select a voice from your available voice clones
3. Click **Generate Voiceover**
4. Preview the audio using the player
5. If satisfied, proceed to Step 4

### Step 4: Generate Video

1. Review your selections on the **Step 4: Video** page
2. Click **Generate Video**
3. Wait for the video to process (usually 1-2 minutes)
4. Once complete:
   - Preview the video
   - **Download** to save to your device
   - **Save to Projects** to keep in your library

## Managing Your Projects

### Viewing Projects
- Click **My Projects** in the navigation
- See all your saved video projects
- Each project shows: thumbnail, name, date, and script preview

### Resuming a Project
- Click **Resume** on any project card
- You'll be taken to the appropriate step
- Continue where you left off

### Exporting Projects
- Click the export icon on a project card
- A JSON file will download with your project data
- Use this to backup your work

### Deleting Projects
- Click the trash icon on a project card
- Confirm deletion
- Note: This cannot be undone

## Managing Your Scenes

Scenes are saved automatically when generated. To manage them:

1. Go to **Step 2: Scene**
2. Browse the **My Scenes** tab
3. Delete scenes you no longer need by clicking the trash icon

## Tips for Best Results

### Script Tips
- **Length**: 30-60 seconds of speech works best
- **Tone**: Write conversationally, as if talking to a friend
- **Punctuation**: Use commas for short pauses, periods for longer ones
- **Numbers**: Write out numbers (e.g., "twenty-five" not "25")

### Scene Generation Tips
- **Reference Images**: Use clear, well-lit photos
- **Prompts**: Be specific about clothing, setting, and lighting
- **Orientation**: Match your intended platform (vertical for social, horizontal for presentations)

### Voiceover Tips
- **Voice Selection**: Choose a voice that matches your content tone
- **Review**: Always preview before generating video
- **Regenerate**: If it doesn't sound right, try regenerating

## Troubleshooting

### "Failed to generate scene"
- Check your Kie.ai API key in Settings
- Ensure you have available credits
- Try a simpler prompt

### "Failed to load voices"
- Check your ElevenLabs API key in Settings
- Ensure you have voice clones created in ElevenLabs

### "Video generation timed out"
- Long scripts may take more time
- Try with a shorter script first
- Check your internet connection

### Data not saving
- Your browser may be blocking storage
- Check if you're in private/incognito mode
- Ensure you have available storage space

### Clear all data
- Open Settings
- Scroll to "Danger Zone"
- Click "Clear All Data"
- Confirm the action
